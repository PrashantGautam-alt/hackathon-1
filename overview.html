AI Agent Framework with Q-Learning

Built using JavaScript

By Prashant Gautam, Lavanya Singh, Arpit Saini

Introduction

  • This project demonstrates an AI Agent Framework.
  • It has multiple agents interacting in an environment.
  • One agent uses Q-learning to learn from rewards.

Agent Class

  • Represents a basic agent in the environment.
  • Has a name and can perform actions.
  • Example: 'Agent 1 is acting.'

Environment Class

  • Manages multiple agents in the environment.
  • Can add agents and simulate their actions.
  • Uses a loop to call each agent's act() function.

Q-Learning Agent

  • Extends the Agent class.
  • Uses a Q-Table to store state-action-reward values.
  • Can learn from rewards and improve actions.

Code Flow

  1. Create Environment.
  2. Create Agent and Q-Learning Agent.
  3. Add agents to environment.
  4. Simulate environment.
  5. Train Q-Learning Agent.

Expected Output

Agent 1 is acting.
Q-Agent is acting.

Training Q-Agent...
Q-Table: { 'state1-action1': 10 }
